//图标字体 device-width - viewport的宽度 height - viewport的高度 device-width 为设备的宽度（单位为缩放为 100% 时的 CSS 的像素） initial-scale - 初始的缩放比例 minimum-scale - 允许用户缩放到的最小比例 maximum-scale - 允许用户缩放到的最大比例 target-densitydpi=high-dpi,(设置屏幕密度为高频，中频，低频自动缩放) user-scalable - 用户是否可以手动缩放 工作中遇到这样的一个问题：当用户获取弹框的 input 焦点时， 在Android手机的浏览器是正常的，可是在苹果手机上的浏览器中就会自动放大整个网页; 解决的方法就是，当用户获取焦点后就设置user-scalable=no,当用户失去焦点后user-scalable=yes; 具体的viewport解释看下面===========> 原文：http://www.php100.com/html/webkaifa/HTML5/2012/0831/10979.html
